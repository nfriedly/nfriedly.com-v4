FB.getLoginStatus example with alcohol restriction

This page initializes the Fb JS SDK with app ID 113630655336080, which has the Alcohol restriction enabled.

If you click the button, it should alert your current login status (one of unknown, not_authorized, or connected). However, for users who are not logged into FB (unknown), it fails to alert anything.

If you check the network traffic for the unknown case, you can see the response to http://www.facebook.com/dialog/oauth?... is <span>Application Error: There was a problem getting data for the application you requested. The application may not be valid, or there may be a temporary glitch. Please try again later.</span>
