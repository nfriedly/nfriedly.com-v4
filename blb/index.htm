Blue Letter Bible Search Plugin (KJV)

This page allows you to add a Blue Letter Bible search option to your web browser's built in search bar for quicker access.

  • Install for FireFox
  • Install for Internet Explorer 7
  • How to use it

How to install in FireFox

In FireFox you should have a search bar at the top with a down arrow. Click it. There will be an option that says Add "Blue Letter Bible KJV Search". Click that.

How to install in Internet Explorer 7

In Internet Explorer 7 it's almost as easy. You click the down arrow again, and it fakes you out and pretends that it's already installed. Ignore the stared "Blue Letter Bible KJV Search" and instead go to the Add Search Providers option. Another Blue Letter Bible KJV Search should pop out of the side, click on that one.

How to use it

You can type a scripture reference (such as "2 timothy 2:15" or "2 tim 2:15") and it will take you to the chapter; if you specify a verse it will jump down the page to that verse. Or you can type in your search terms (such as "study truth") to get a listing of all of the verses containing those words.

For more details and advanced tips, visit the Blue Letter Bible's How to Use the Bible Search.

Congratulations, you're done!
